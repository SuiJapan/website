# Sui Japan Community Web Site

## CI/CD Pipeline

- GitHub Actions

## How to work

- install

    ```bash
    pnpm i
    ```

- formatter & linter

    ```bash
    pnpm run format
    ```

    ```bash
    pnpm run lint
    ```

- build

    ```bash
    pnpm run build
    ```

- start on local

    ```bash
    pnpm run dev
    ```
